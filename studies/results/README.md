# Curated Experiment Results

This is the canonical home for self-contained experiment bundles.

Each bundle should contain:

- `procedure/`: how the experiment was run
- `inputs/`: workload files or symlinks to the workload files
- `raw/`: preserved raw run slices, plans, and synced analysis fragments
- `metrics/`: derived CSV/JSON summaries
- `graphs/`: rendered plots
- `archive/`: redundant or intermediate artifacts kept only for safety

The compatibility alias:

- `results/experiments/`

points here.
